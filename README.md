Descramble
==========

## Requirements

* wxPython

## Usage

```bash
python ./descramble.py "H**ENCS++IMHN++ORASP++EN"
```

* + = double letter
* ++ = triple letter
* \* = double word
* \*\* = triple word
* NOTE: The 'Qu' tile should be expressed as 'Q'
