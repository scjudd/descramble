Scramble Solver
===============

## Usage

To output results to the terminal:

```bash
python ./scramble.py "H**ENCS++IMHN++ORASP++EN"
```

To use the GUI (requires wxPython):

```bash
python ./gui.py "H**ENCS++IMHN++ORASP++EN"
```

* + = double letter
* ++ = triple letter
* \* = double word
* \*\* = triple word
* NOTE: The 'Qu' tile should be expressed as 'Q'
